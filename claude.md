# Smart Guide ‚Äî Banking Product Recommendation Engine

## Project Type
Existing POC being upgraded to production quality. All changes must be incremental
and non-breaking ‚Äî the current working system must remain functional at every step.

## Current Architecture
- **Main API**: Spring Boot 3.2, Java 17, Maven, port 8080
- **Scraper Service**: Separate Spring Boot app, Playwright, port 8081
- **Frontend**: React 18 + Create React App (CRA) + Material UI (MUI), served from static resources
- **Database**: PostgreSQL 15+ with Flyway migrations
- **LLM**: Azure OpenAI or Ollama (configurable via app.llm.provider)
